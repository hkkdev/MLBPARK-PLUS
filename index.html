MLBPARK PLUS

MLBPARK PLUS는 미섹사MLBPARK를 좀 더 깔끔하고 속 편하게 이용할 수 있도록 도와주는 크롬 확장프로그램입니다.

주요 기능

  • 컨테이너 너비 확장, 조절 기능
  • 제목 아이콘 표시 기능
  • 제목 차단 기능
  • 닉네임 차단 기능
  • 본문 내용 가리기 기능
  • 글 작성자의 최신글 표시 기능
  • 회원 비밀번호 변경 요청 무시 기능
  • 자체 오류 수정 및 성능 개선 등

필요사항

MLBPARK PLUS를 이용하기 위해서는 구글 크롬이 설치되어 있어야 합니다. 현재 모바일 버전 크롬은 확장 프로그램이 지원되지 않아 설치하실 수 없습니다.

설치 방법

MLBPARK PLUS는 구글 웹스토어에서 설치 버튼을 클릭하여 설치 가능합니다. https://chrome.google.com/webstore/detail/migdlkcljdaobpodfbpcdpihehhhjefm 크롬 브라우저에서 위 URL로 접속하신 뒤 설치 버튼을 클릭해주세요.
